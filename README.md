# Linux Log File Analysis, Automation, and SIEM Visualization

## Overview

This project demonstrates log analysis and security monitoring techniques using Linux authentication logs, automated analysis methods, and SIEM visualization tools. The objective of the project was to identify suspicious login activity, analyze authentication events, and understand how security analysts monitor and investigate potential threats within a Linux environment.

The project combined manual log review, automated analysis techniques, and SIEM-based visualization to simulate real-world security operations workflows commonly used in cybersecurity and SOC environments.

## Tools Used

* Linux Authentication Logs
* Splunk SIEM
* Command-Line Log Analysis Utilities

## Skills Demonstrated

* Log Analysis
* Security Monitoring
* Authentication Event Analysis
* Incident Detection
* SIEM Visualization
* Threat Investigation
* Linux Security Fundamentals
* Security Operations (SOC) Concepts

## Key Activities

* Reviewed Linux authentication logs to identify failed login attempts and suspicious activity
* Analyzed authentication failures, invalid user attempts, and login patterns
* Performed automated log analysis techniques to improve event investigation efficiency
* Visualized security events and trends using Splunk dashboards and searches
* Investigated indicators of potential unauthorized access attempts

## Key Findings

* Multiple failed authentication attempts were identified
* Invalid user login activity was detected within the logs
* Log analysis provided visibility into suspicious login behavior and authentication trends
* SIEM visualization improved monitoring and event correlation capabilities

## Conclusion

This project demonstrates practical experience with log analysis, security monitoring, and SIEM workflows used in cybersecurity operations. It highlights the importance of centralized logging, event analysis, and visibility into authentication activity for detecting and responding to potential security incidents.

## Disclaimer

This project was conducted in a controlled and authorized lab environment for educational and cybersecurity training purposes only.
